Authors: Joseph Lenton - JL235@Kent.ac.uk.

Description

A random number generator server.

You create a new random server by calling the new function and then pass this into the random function when generating numbers.

This is only meant as a cheap and cheerful random number module. For anything more sophisticated use the random module supplied in the Erlang OTP.
