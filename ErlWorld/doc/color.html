Defines how to create colours and how to retrieve the values from colours.

Authors: Joseph Lenton.

Description

Defines how to create colours and how to retrieve the values from colours. Colours are stored as a tuple in the form {red, green, blue, alpha}. All components are floats.

By using this file it allows the user to abstract away how they care about colours.

It also contains common colours for you to use.
