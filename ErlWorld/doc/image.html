Description

This is for handling image loading and defines the structure of an image.

It offers functions for also retrieving the properties of an image, such as it's size.

Internally this image actually gets the given graphics device to create the image. This is because only the graphics' process is able to safely make the WX and OpenGL calls needed to create the image.
