for/2

for(Max::integer(), Func::(I::integer()) -> term()) -> Count::integer()

The for loop applies a function to each value in the loop in turn, until the value reaches max. This defaults the start of the loop to 0 and steps by 1 between each value.

for/3

for(Max::integer(), Func::(I::integer()) -> term(), Step::integer()) -> Count::integer()

This for loop will iterate from 0 to max, but increments the value in the iterator by Step amount each time.

for/4

for(Min::integer(), Max::integer(), Func::(I::integer()) -> term(), Step::integer()) -> integer()

This for loop will iterate from min to max where min is inclusive and max is exclusive. It will increment by the value in Step for each iteration.

foreach_count/2
